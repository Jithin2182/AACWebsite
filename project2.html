Content Based Image Retrieval

Dr. K. Meenakshi

Content based image retrieval is based on matching the features of query image with that of image .Content based means that the search analyzes the content of the image rather than the metadata such as keyword tags or descriptions associated with the image. It may require long processing due to the amount of images to be analyzed in a database, so comparision between images is made using a set of features extracted from the images. The implementation of such a system requires the extraction and storing of the image features to be compared with the features of the query image. With this flow, the implementation process is more dynamic, since all features have already been stored somewhere.
